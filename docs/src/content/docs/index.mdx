## What is Claimo?

Claimo lets players redeem **reward codes** with a configurable command. Each code runs
one or more server commands and can be gated behind **requirements** — blocks mined,
playtime, account age, and more. The requirement system is a public API, so addons can
plug in brand-new checks such as *"follow us on TikTok / YouTube / X / Instagram"*.
